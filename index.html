Java IO流知识笔记

1 流的分类

IO流主要用于对计算机的硬板、内存、键盘等处理设备上的数据进行操作，说的通俗一点就是用来对文件内容进行操作的。

按流的方向分为：输入流和输出流。
按数据类型分为：字节流和字符流。
按流的功能分为：节点流和处理流。

字符流和字节流的区别：
首先熟悉一些概念：
字节：也就是Byte，它是通过网络传输信息或信息存储的单位。字节是计算机信息技术用于计量存储容量和传输容量的一种计量单位，1个字节等于8位二进制，它是一个8位的二进制数，是一个很具体的存储空间。
字符： 字符是指计算机中使用的字母、数字、字和符号，是人们使用的记号，抽象意义上的一个符号。包括：
1、2、3、A、B、C、~！·#￥%……—（）——+
等等。
*首先程序中的数据都是以流的形式保存的，而流中保存实际上都是字节文件。
